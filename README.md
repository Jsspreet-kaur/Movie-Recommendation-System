# **Movie-Recommendation-System** - Movie Recommendation System using content-based filtering and cosine similarity
 
The project is a **Content-Based Recommendation System** developed using Python and other Technologies. It recommends the mivies based on the reference by the user selected movie and calculating similarity using the **Cosine-Similarity** algorithm.
The application provides an interactive user interface built with **Streamlit**, allowing usersto search for a movieand receive personalized recommendations instantly.

**Features**
 1. Content-Based movie recommendation
 2. interactive streamlit web application
 3. Fast and accurate moviesuggestions
 4. User-friendly interface
 5. Similarity calculation using Cosine-Similarity

**Technologies Used**
1. Python
2. Pandas
3. NumPy
4. Scikit-learn
5. Streamlit
6. Pickle

**To open Movie-recommendation web app-** 
   streamlit run apptask2SKT.py

   apptask2SKT.py- It is the name given by user for the python file. User can change to any name with .py extension.
